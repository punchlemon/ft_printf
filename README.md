# ft_printf
This is Project to recreate the function 'printf'.
## Installation
```bash
git clone https://github.com/punchlemon/ft_printf
```
```bash
cd ft_printf
```
```bash
git submodule init
```
```bash
git submodule update --remote --recursive
```
## Test
```bash
cd test_ft_printf
```
```bash
make ; ./test
```
